# Will Power Excavation - Resource Planning Dashboard

A simple, single-file React demo for scheduling crews and equipment across different project stages.

## Features

- **Login Protection**: Simple password protection (password: `1234`)
- **Schedule Board**: Drag & drop interface for scheduling crews and equipment
  - Left panel: List of crews and equipment (draggable)
  - Right panel: 7-day horizontal calendar view with job rows
- **Job Stages**: Display and manage project stages (Clearing, Utilities, Grading, Final Prep)
- **Edit Job Modal**: Edit job address and stage
- **Multiple Views**: Schedule Board, Crew Management, Equipment, and Active Jobs

## How to Use

1. Simply open `index.html` in your web browser
2. Login with password: `1234`
3. Start scheduling!

## Files

- `index.html` - Main application file (includes all React code)
- `style.css` - External stylesheet
